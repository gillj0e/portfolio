The Phish Bowl

Internship Capstone Project

Project Details:

A full-stack web application that gamifies phishing-awareness training for employees. Built with Python (Flask), JavaScript, HTML/CSS, and SQL, the platform challenges users to identify phishing emails through timed rounds and instant feedback. It features user authentication, score tracking, leaderboards, and an admin dashboard for monitoring engagement. Hosted on Microsoft Azure, Phish Bowl combines cybersecurity education with interactive gameplay to make learning about digital safety engaging and effective.

Project Status: Demo Completed
